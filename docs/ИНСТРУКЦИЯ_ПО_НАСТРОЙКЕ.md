# Инструкция по настройке (БЕЗ Google Drive API)

Приложение теперь работает **БЕЗ настройки Google Drive API**! 

Вам нужно только получить прямые ссылки на файлы или их ID.

## Шаг 1: Получение ссылок на файлы

Для каждого файла, который нужно скачать:

1. Откройте Google Drive и найдите файл:
   - `Испп 23-09.2.xslm`
   - `Испп 23-09.1.xslm`
   - `Испп 24-11.xslm`

2. Правой кнопкой мыши на файл → **"Получить ссылку"** (или "Share")

3. Убедитесь, что доступ установлен как **"Любой, у кого есть ссылка"** (или "Anyone with the link")

4. Скопируйте ссылку. Она будет выглядеть так:
   ```
   https://drive.google.com/file/d/1ABC123xyz456DEF789/view?usp=sharing
   ```

5. Из этой ссылки нужно извлечь ID файла (часть между `/d/` и `/view`):
   - В примере выше ID: `1ABC123xyz456DEF789`

## Шаг 2: Настройка config.py

Откройте файл `config.py` и найдите секцию с настройками.

### Вариант А: Использовать ID файлов (проще)

Найдите `FILE_IDS` и укажите ID каждого файла:

```python
FILE_IDS = [
    "1ABC123xyz456DEF789",  # ID файла Испп 23-09.2.xslm
    "1XYZ789abc123DEF456",  # ID файла Испп 23-09.1.xslm
    "1QWE456rtz789YUI012",  # ID файла Испп 24-11.xslm
]
```

### Вариант Б: Использовать прямые ссылки

Найдите `DOWNLOAD_LINKS` и укажите прямые ссылки на скачивание:

```python
DOWNLOAD_LINKS = [
    "https://drive.google.com/uc?export=download&id=1ABC123xyz456DEF789",  # Испп 23-09.2.xslm
    "https://drive.google.com/uc?export=download&id=1XYZ789abc123DEF456",  # Испп 23-09.1.xslm
    "https://drive.google.com/uc?export=download&id=1QWE456rtz789YUI012",  # Испп 24-11.xslm
]
```

**Формат ссылки:** `https://drive.google.com/uc?export=download&id=FILE_ID`

Где `FILE_ID` - это ID файла из шага 1.

## Важно!

- Файлы должны быть доступны по ссылке (права доступа: "Любой, у кого есть ссылка")
- Порядок файлов в списке должен соответствовать порядку в `TARGET_FILES`
- Если файл не скачивается, проверьте права доступа к файлу на Google Drive

## Решение проблем

**Проблема:** Файл не скачивается
- Проверьте, что файл доступен по ссылке
- Убедитесь, что ID файла указан правильно
- Попробуйте использовать прямую ссылку вместо ID

**Проблема:** Ошибка при скачивании
- Убедитесь, что файл не слишком большой
- Проверьте интернет-соединение
- Попробуйте открыть ссылку в браузере - если она работает, значит ID правильный
